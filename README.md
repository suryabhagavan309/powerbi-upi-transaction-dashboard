# UPI Transaction Analysis Dashboard

## 📊 Overview
This project analyzes UPI transaction data using Power BI to understand transaction trends, balances, and user behavior.

## 🎯 Key Features
- Monthly transaction trend analysis
- Dynamic chart switching (Line & Column)
- Balance vs Amount comparison
- Interactive filters (Bank, City, Device, Gender)
- Detailed transaction-level table

## 🛠 Tools Used
- Power BI
- Data Modeling
- Data Visualization
